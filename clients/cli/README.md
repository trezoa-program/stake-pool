# TPL Stake Pool program command-line utility

A basic command-line for creating and using TPL Stake Pools.  See https://tpl.trezoa.com/stake-pool for more details.

Under `./scripts`, there are helpful Bash scripts for setting up and running a
stake pool. More information at the
[stake pool quick start guide](https://tpl.trezoa.com/stake-pool/quickstart).
